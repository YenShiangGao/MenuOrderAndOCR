# 後台操作手冊

這份是給**餐廳老闆 / 店長 / 員工**看的後台操作說明,用最白話的方式介紹「怎麼用」這套系統。

不需要任何電腦技術背景,照著做就會了。

---

## 後台網址

> **https://menu-order-and-ocr.vercel.app/login**

可以加到瀏覽器書籤,以後直接點開。

也可以從手機開,但因為要打字比較多、要選圖片,建議**用桌機或筆電**操作。

---

## 第一次登入

收到工程師寄來的帳號密碼後:

1. 打開上面的網址
2. 輸入 **Email**(例如 `owner@yourrestaurant.com`)
3. 輸入**密碼**
4. 點「**登入**」

登入成功後,左邊會出現一條黑色的選單列,顯示三個項目:

- **首頁**(目前在哪個分類有幾道菜)
- **菜單分類**
- **菜品**

---

## 後台首頁長什麼樣子

進入後台後第一眼會看到三個方塊:

| 卡片 | 意思 |
|---|---|
| 啟用中分類 | 目前**有開放給客人看到**的分類數量 |
| 上架菜品 | 目前**有開放給客人看到**的菜品數量 |
| 今日售完 | 今天標示為「售完」的菜品數量(隔天 0 點會自動歸 0) |

---

## 第一步:先建分類

「分類」就是菜單的分組,例如:**主餐 / 小菜 / 飲料 / 甜點**。

客人看到的菜單會按你設定的分類,一塊一塊呈現。

### 怎麼新增分類

1. 左側選單點「**菜單分類**」
2. 右上角點藍色的「**新增分類**」按鈕
3. 跳出視窗 → 輸入分類名稱(例如「主餐」)
4. 點「**新增**」
5. 表格就會多一列,右上會跳「已新增分類」的提示

### 怎麼改分類名字

1. 該列右側的**鉛筆**圖示
2. 改名字 → 點「**更新**」

### 怎麼暫時停用某個分類(例如「夏季冰品」冬天先收起來)

1. 該列「**狀態**」欄位的**綠色圓角標籤**(寫「啟用」)
2. 點一下 → 會變灰色「停用」
3. 客人那邊**立刻看不到**這個分類

要恢復就再點一次,會變回綠色。

### 怎麼刪除分類

1. 該列右側的**垃圾桶**圖示
2. 跳出確認視窗 → 點「**刪除**」

⚠️ **如果這個分類底下還有菜品,系統不會讓你刪**,會跳紅字提醒。
要先把那些菜品**改到別的分類**或**直接刪掉菜品**,才能刪分類。

---

## 第二步:新增菜品

### 怎麼新增菜品

1. 左側選單點「**菜品**」
2. 右上角點「**新增菜品**」
3. 跳出視窗,填:

   | 欄位 | 說明 |
   |---|---|
   | 品名 | 必填,例如「招牌牛肉麵」 |
   | 分類 | 必填,從下拉選單選 |
   | 價格 | 必填,**只填數字**(不要寫「元」或「$」) |
   | 描述 | 選填,介紹食材或特色,80 字內 |
   | 圖片 | 選填,點「選擇圖片」上傳一張照片 |

4. 點「**新增**」

### 圖片要求

- 格式:**JPG / PNG / WebP**(手機拍的 HEIC 要先轉檔)
- 大小:**5 MB 以內**(一般手機照片是 2-4 MB,通常 OK)
- 比例:**正方形最好看**(例如 1000x1000),非正方形也能傳但會被裁切顯示
- 拍法建議:**俯拍 45 度**、白色或木質桌面、自然光

選完圖會立刻顯示**預覽**。覺得不喜歡:

- 點預覽右上的 **X** 把它移除
- 或點下面的「**換一張**」重選

### 編輯菜品

1. 該列右側的**鉛筆**圖示
2. 想改什麼就改什麼,然後點「**更新**」
3. 編輯模式比新增模式多一個「上架」開關(下面會說)

### 換圖 / 移除圖

進入編輯後,如果該菜品已經有圖,圖會出現在預覽區:

- **換新圖**:點「換一張」→ 選新檔案
- **完全不要圖**:點預覽右上的 **X** → 圖片區會變回「選擇圖片」按鈕 → 直接點更新就會清空

舊圖會自動從伺服器刪掉,不會佔空間。

## 上下架 vs 今日售完(很容易搞混)

這兩個東西**意思不一樣**:

|  | 上架 / 下架 | 今日售完 |
|---|---|---|
| 用在哪 | 永久性,例如菜單改版、菜品退場 | 當日臨時,例如食材用完、暫停供應 |
| 怎麼操作 | 菜品列「**上架**」綠色標籤 → 點變灰色「下架」 | 菜品列「**供應中**」灰色標籤 → 點變橘色「售完」 |
| 客人看到的 | **完全消失** | 仍然出現,但有刪除線 + 橘色「今日售完」標籤 |
| 隔天會怎樣 | 維持下架,直到你手動上架 | **自動恢復**為「供應中」(過 0 點) |

**簡單記憶**:今天賣完了 → 點「售完」;這道菜要長期收掉 → 點「下架」。

---

## 客人那邊看到的菜單

客戶網站的菜單在 **https://menu-order-and-ocr.vercel.app/menu**

有幾個小規則要記:

1. 後台改完之後,**最多 60 秒**客人那邊會看到更新
2. 如果你急著確認,在客戶網站按 **Cmd+Shift+R**(Mac)或 **Ctrl+Shift+R**(Windows)強制重整
3. **沒有上架菜品的分類會整個消失**(例如「甜點」分類底下沒任何上架菜品 → 客人不會看到「甜點」這個標題)
4. **完全沒有任何菜品時**,客人看到的是「菜單尚未建立,請稍候」

## 登出

左下角「**登出**」按鈕,點下去就會回到登入頁。

> **建議習慣**:在公用電腦上操作完一定要登出,免得別人不小心改到菜單。

## 常見問題

### Q:我把密碼忘了怎麼辦?

目前後台**還沒有「忘記密碼」自助功能**,需要請工程師幫你重設。
傳訊息給工程師,告訴他你的 Email,他重設後會給你新密碼。

### Q:為什麼點刪除分類沒反應?

最可能是**底下還有菜品**。系統會跳紅字提醒。
解法:先去「菜品」頁,把那些菜品要嘛刪掉,要嘛改到別的分類,然後再回來刪分類。

### Q:圖片上傳失敗?

幾種可能:

| 訊息 | 原因 |
|---|---|
| 「圖片不可超過 5MB」 | 檔案太大,用手機修圖 app 縮一下 |
| 「只支援 JPG / PNG / WebP」 | 通常是 iPhone 預設的 HEIC,先轉檔 |
| 沒訊息但卡很久 | 網路問題,刷新重試 |

### Q:我手機可以用後台嗎?

**可以,但不太方便**。後台是設計給**桌機**用的,手機螢幕會有點擠。
如果只是要切「今日售完」這種快速動作可以,要新增菜品建議還是用電腦。

### Q:有個菜品我不要它出現在菜單上,可是又不想刪除歷史

點該列的「**上架**」綠色標籤 → 變成灰色「下架」就好。
資料還在,可以隨時再切回上架。

### Q:可以**多人**同時登入嗎?

可以,只是大家**用同一個帳號**。如果想讓不同店員有不同帳號,告訴工程師加。

## 出問題該找誰

工程師 / 客服窗口:**[請工程師交付時填上]**

回報問題請附:

1. 你在哪個頁面(網址截圖)
2. 你做了什麼動作
3. 預期看到什麼
4. 實際看到什麼(有錯誤訊息或紅字也截圖)

這樣最快解決。
